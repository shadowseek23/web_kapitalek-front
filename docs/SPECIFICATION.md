# Specifikace projektu

## Název projektu
Kapitálek – webová prezentace

## Popis
Moderní webová aplikace pro prezentaci služeb, portfolia a kontaktu společnosti Kapitálek. Postaveno na Next.js s využitím TypeScriptu, Tailwind CSS a dalších moderních knihoven.

## Hlavní funkce
- Prezentace služeb a portfolia
- Kontaktní formulář
- Blogová sekce
- Responzivní design
- Tmavý/světlý režim
- Optimalizace pro SEO
- Správa favicon a metadat

## Technologie
- Next.js (React, SSR, SSG)
- TypeScript
- Tailwind CSS
- PostCSS
- @heroui/* komponenty
- Další viz TECHNOLOGIES.md

## Cílová skupina
Zákazníci a partneři společnosti Kapitálek hledající informace o službách, referencích a možnostech kontaktu.

## Nasazení
Aplikace je připravena pro nasazení na Vercel nebo jiný Node.js hosting.
